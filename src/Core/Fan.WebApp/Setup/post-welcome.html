A few tips to get you started.

  • Go to Settings to enter your Disqus and Google Analytics information.
  • Go to Widgets to update Social Icons to your own media links.
  • The appsettings.json files have database, storage, preferred domain and logging settings for dev and prod.
  • When an invalid URL is accessed the 404 page is only displayed in prod, in dev the developer exception page is shown.
  • To run the app with prod settings locally, set ASPNETCORE_ENVIRONMENT to Production. Be aware by default production uses Blob Storage for storing images.
